# Redução de Dimensionalidade em Imagens para Redes Neurais

Este projeto mostra como pré-processar imagens aplicando:
- **Tons de cinza** (0–255)
- **Binarização** (0 ou 255)

Um bom ponto de partida para aprendizado e redes neurais.
